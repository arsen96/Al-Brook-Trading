Un Spike and Channel comporte toujours deux phases.  
→ Spike.  
→ Puis Channel.  
──────  
Phase 1 : Spike.  
→ Mouvement rapide.  
→ Forte urgence.  
→ Peu ou pas de pullbacks.  
→ Les traders sont convaincus que le marché ira encore plus loin.  
→ Le Spike est un Breakout.  
→ Le Spike est aussi un Gap.  
→ Le Spike est aussi un Climax.  
──────  
Après le Spike.  
→ Il y a toujours une pause ou un pullback.  
→ Il peut durer 1 bougie ou plusieurs dizaines de bougies.  
──────  
Phase 2 : Channel.  
→ La tendance continue.  
→ Mais avec moins de momentum.  
→ Plus de pullbacks.  
→ Plus de recouvrement entre les bougies.  
→ Plus d'incertitude.  
→ Plus de Two-Sided Trading.  
──────  
Pendant le Channel.  
→ Trader principalement dans le sens de la tendance.  
→ Les contre-tendances sont souvent des pièges.  
──────  
Le Channel termine souvent.  
→ Sur un Measured Move.  
→ Ou sur une troisième poussée (Third Push).  
──────  
Quand le Channel casse contre la tendance.  
→ Ne pas entrer immédiatement.  
→ Attendre le Pullback.  
→ Exemple :  
→ Bull Channel cassé vers le bas.  
→ Attendre un Lower High. comme par exemple un DT
→ Puis vendre.  
──────  
Après la cassure du Channel.  
→ Le marché revient souvent tester le début du Channel.  
→ C'est aussi un test du Gap créé par le Spike.
